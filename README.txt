Installation Guides
1. This project is using Expo CLI which is a lightweight React Native application tool.
2. Node.js  is needed to execute and "Expo Go" can be downloaded app from Google Play Store for developers.
3. If npm/npx give errors, create a new file named npm manually under the pathway 'C:\Users\*****(username)\AppData\Roaming'
4. The project can be debugged and viewed synchronously on a mobile device by command 'npx expo start' and scan the QR code with the device.
